# ระบบแสกนเอกสารด้วยWebcam

### แนวคิด
มาจากการที่อยากจะลดระยะเวลาทำงานของเจ้าหน้าที่จากเดิมที่ต้อง 
แกะเอกสาร > เรียงเอกสาร > นำเอกสารเข้าเครื่อง > Scan > บันทึกข้อมูลเข้า DB > เรียงกลับคืนแฟ้มผู้ป่วย 

ให้เหลือแค่
ถ่ายรูป > ยัดเข้า DB จบ!!!
